# blue-velvet-page
Página de fã sobre o filme Veludo Azul (1986) de David Lynch, feita com HTML e CSS. 

Contém:
- Informações sobre o filme
- Sinopse e elenco
- Trailer incorporado
- Avaliação pessoal
- Formulário de comentários

Tecnologias usadas:
- HTML5
- CSS3
- Google Fonts
